# Geiger Notes Market Research

Last updated: June 2026

## Goal

Geiger Notes is a visual, free-form canvas and board-based notes application. This research file tracks current leading notes apps, what they offer, and which features are common, differentiated, or rare enough to influence product positioning and pricing.

The immediate goal is to build a feature/function inventory, rank feature frequency across competitors, then choose the right feature set for Geiger Notes based on relevance, implementation cost, and price competitiveness.

## Milanote Capability Target

If "mila notes" refers to Milanote, its capability set is especially relevant because Geiger Notes is already positioned around canvas/board notes.

Core Milanote-style capabilities:

- Freeform boards
- Drag-and-drop mixed content cards
- Notes, images, links, videos, files, sketches, documents, and tables
- Columns for grouping and lightweight workflow
- Nested boards
- Templates
- Web clipper
- To-do lists
- Due dates, assignees, and reminders
- Real-time collaboration
- Comments
- Mentions
- Reactions
- Read-only and comment-only sharing
- Board sharing/presentation
- Export options

## Feature Frequency Buckets

### Very Common / Table Stakes

These features are expected in almost every serious notes product.

- Rich text notes
- Folders, notebooks, or spaces
- Tags or labels
- Search
- Attachments and images
- Web, desktop, or mobile access
- Cloud sync
- Dark mode
- Basic sharing
- Export

### Common Premium

These are widely available, often used to justify paid plans.

- Web clipper
- OCR for images and PDFs
- Offline access
- Reminders
- Tasks and checklists
- Templates
- Version history
- PDF/document export
- Collaboration comments
- Advanced search
- AI summarization
- AI rewrite/editing

### Modern Differentiators

These are common among modern PKM or workspace products but not universal.

- Backlinks
- Graph view
- Block editor
- Markdown/local-first storage
- Canvas or whiteboard mode
- Publish-to-web
- Databases and tables
- Object types or structured note types
- Daily notes
- End-to-end encryption
- Plugins
- API or automation support

### Rare / High-Signal

These features can create strong differentiation if implemented well.

- Milanote-style visual boards with mixed media and project workflow
- AI chat over personal notes
- Meeting transcription
- AI-generated action items
- Custom AI agents
- Supertags or object schemas
- PDF, YouTube, or source annotation
- MCP or agent access
- Self-hosting with sync and collaboration
- Custom AI provider support or bring-your-own-key AI

## Early Positioning Hypothesis

Geiger Notes should not try to beat every notes app at everything. The strongest path appears to be:

> Milanote-style visual boards, but more affordable, faster for personal knowledge and project planning, and eventually smarter with AI/search.

This means the first serious feature matrix should favor:

- Visual board and canvas depth
- Fast capture and organization
- Useful project/task structure
- Sharing and export
- Search across visual content
- Price-sensitive premium packaging
- AI only where it directly improves note management
